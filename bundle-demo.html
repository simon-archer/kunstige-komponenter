How to Use

To use this bundle in your own project:

<!-- Import all components with a single import -->
<script type="module">
  import bundleInfo from 'https://cdn.jsdelivr.net/gh/yourusername/kunstige-komponenter@main/simple-bundle.js';
  console.log(`Loaded ${bundleInfo.name} v${bundleInfo.version}`);
</script>

<!-- Then use any component -->
<ki-message-field placeholder="Type a message..."></ki-message-field>

Important: Replace yourusername with your actual GitHub username.
