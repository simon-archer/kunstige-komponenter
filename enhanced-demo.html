How to Use

To use this enhanced bundle in your own project, simply import it:

<!-- Import the enhanced bundle -->
<script type="module">
  import bundleInfo from 'https://cdn.jsdelivr.net/gh/yourusername/kunstige-komponenter@main/enhanced-bundle.js';
  console.log(`Loaded ${bundleInfo.name} v${bundleInfo.version}`);
  console.log(`Available icons: ${bundleInfo.svgIcons.join(', ')}`);
</script>

<!-- Then use any component -->
<ki-message-field placeholder="Type a message..."></ki-message-field>

Important: Replace yourusername with your actual GitHub username.
