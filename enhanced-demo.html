How Auto-Discovery Works

The bundle now automatically:

  1. Discovers all components in the component directories
  2. Analyzes each component to determine its dependencies
  3. Sorts components based on their dependencies (topological sort)
  4. Loads components in the correct order

No hardcoded lists or manual registration required!

How to Use

To use this enhanced bundle in your own project, simply import it:

<!-- Import the enhanced bundle -->
<script type="module">
  import KiComponents from 'https://cdn.jsdelivr.net/gh/yourusername/kunstige-komponenter@main/bundle.js';
  
  // Wait for components to be ready
  KiComponents.ready(({ isReady, components }) => {
    console.log(`Loaded ${components.length} components successfully:`, components);
    
    if (isReady) {
      // Do something with the loaded components
    }
  });
</script>

<!-- Then use any component -->
<ki-message-field placeholder="Type a message..."></ki-message-field>

Important: Replace yourusername with your actual GitHub username.
